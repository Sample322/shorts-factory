Privacy Policy

Effective date: June 19, 2026

Overview

Shorts Factory is a local tool used by a creator to prepare short-form videos and publish finished clips to connected creator accounts. The public website is used for app information, policy pages, and OAuth redirect support. It does not provide a hosted video editing backend.

Information processed locally

The local app may process files selected by the user, generated clips, transcription text, clip metadata, captions, tags, OAuth tokens, and platform upload status. This data is stored on the user's own computer unless the user chooses to upload a finished clip to a connected service.

TikTok and third-party services

When the user connects TikTok, Shorts Factory uses TikTok Login Kit to receive an OAuth access token for the authorized account. The app uses TikTok Content Posting API only after the user enables TikTok upload. TikTok receives the finished video file, caption, selected privacy setting, and interaction settings needed to process the upload.

Website data

This static website does not intentionally collect personal information, does not run analytics, and does not store user submissions. Hosting providers such as GitHub Pages may process basic server logs according to their own policies.

Data sharing

Shorts Factory does not sell personal information. Data is shared with third-party platforms only when the user authorizes an account and asks the app to upload content to that platform.

User control

The user can remove local OAuth tokens by using the app's reset/revoke controls or deleting the local token cache. The user can also revoke app access from the connected platform account settings.

Contact

For privacy questions, contact the app owner through the GitHub repository or the contact information provided in the TikTok Developer app submission.
