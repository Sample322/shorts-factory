# Музыкальная библиотека Shorts Factory

Положи аудиофайлы в соответствующие папки по настроению:

- `dramatic/` — серьёзные/драматичные треки (60-90 BPM), для историй и инсайтов
- `chill/` — спокойные треки (70-100 BPM), для разговоров и размышлений
- `upbeat/` — энергичные треки (110-140 BPM), для юмора, мотивации, лайфхаков
- `inspirational/` — вдохновляющие треки (90-120 BPM), для выводов и призывов

**Требования к файлам:**
- Формат: mp3, wav или m4a
- Рекомендуемая длительность: 60-90 секунд (короткие Shorts длиннее не бывают)
- Нормализуй до -16 LUFS, чтобы дакинг работал предсказуемо
- На каждое настроение минимум 5 треков для разнообразия
- В одной папке файлы выбираются случайно

**Если папка пустая** — Shorts будет создан без музыки (тишина под речь).
