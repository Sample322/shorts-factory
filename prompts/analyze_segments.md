КРИТИЧЕСКИ ВАЖНО:
- Не выдумывай имена персонажей, названия сериалов/фильмов, события или цитаты.

- Не пытайся делать “киношный трейлер” из сцены. Нужен понятный короткий момент,

- Если точного имени персонажа нет в транскрипте — используй роль по сцене

══════ КАК АЛГОРИТМ YOUTUBE SHORTS 2026 ОЦЕНИВАЕТ ВИДЕО ══════

Алгоритм работает по модели test-and-expand: первые 100-500 зрителей решают судьбу.
Три ключевые метрики (от важнейшей к меньшей):

1. **Swipe-away rate в первые 0.5-2 сек** — самый сильный НЕГАТИВНЫЙ сигнал.
   Если хук слабый, зритель свайпает мимо за 1 секунду = видео мёртвое на 300 просмотрах.

2. **Watch-through rate (досмотр)** — доля досмотревших до конца.
   Чем ближе к 100% (или зацикливание/повторный просмотр) — тем сильнее positive signal.

3. **Loop rate** — пересматривает ли зритель видео по 2-3 раза подряд.
   Это сильнейший indicator качества, алгоритм ОЧЕНЬ его любит.

ВЫВОД: ВЫБИРАЙ моменты, у которых сильное начало И сильный/циклабельный конец.

══════ КРИТЕРИИ ХОРОШЕГО МОМЕНТА (в порядке приоритета) ══════

1. **СИЛЬНЫЙ ХУК В ПЕРВЫЕ 0.5-2 СЕКУНДЫ** (КРИТИЧНО):
   - НА `start` секунде должна СРАЗУ идти яркая реплика, действие или эмоция
   - НЕ ставь start на тишину, паузу, заставку, "так… ну…"
   - Первое слово/фраза должно быть провокацией, вопросом, шокирующим заявлением,
     эмоциональным криком, культовой цитатой, смехом, дракой, конфликтом

2. **ЦЕЛЬНОСТЬ БЕЗ КОНТЕКСТА**:
   - Зритель не знает что было до — клип должен быть понятен сам по себе
   - Мини-история с началом, серединой, концом

3. **ЭМОЦИОНАЛЬНЫЙ ПИК В СЕРЕДИНЕ/КОНЦЕ**:
   - смех, ссора, неловкость, открытие, инсайт, поворот сюжета
   - Должен быть «momentum» — нарастание к развязке

4. **ЦИКЛАБЕЛЬНОСТЬ КОНЦА**:
   - Идеально если конец естественно «возвращает» к началу
     (зритель посмотрел → захотел пересмотреть)
   - Punchline, шокирующая фраза, неожиданный поворот в финале

5. **ЯРКАЯ ЦИТАТА** для title:
   - Хук-цитата которую можно вынести в title

6. **ДЛИТЕЛЬНОСТЬ**: СТРОГО от {min_sec} до {max_sec} секунд (идеал {target_sec}).
   end - start ОБЯЗАНА быть >= {min_sec} и <= {max_sec}.

══════ ЧТО НЕ ВЫБИРАТЬ (СТРОГО ЗАПРЕЩЕНО) ══════

🚫 **ЗАСТАВКА СЕРИАЛА И ТИТРЫ — БАН РОЛИКА!**
   - Признак заставки: в транскрипте ПУСТО или только короткие повторяющиеся
     реплики ("та-да-да", припев, "Рик и Морти", пение интро)
   - YouTube Content ID идеально матчит заставки → автоматический бан
   - Если момент рядом с заставкой, ставь `start` ПОСЛЕ её окончания
   - Если момент рядом с титрами, ставь `end` ДО их начала
   - Постпроцессор автоматически вырежет заставку из клипа, но ТЫ должен
     попытаться выбирать моменты так, чтобы заставка не пересекалась с клипом

🚫 **МОМЕНТЫ БЕЗ ДИАЛОГА**
   - НЕ ВЫБИРАЙ moments где идёт только музыка/SFX без реплик
   - НЕ ВЫБИРАЙ если в клипе < 8 слов суммарно
   - НЕ ВЫБИРАЙ если в первые 3 секунды клипа НЕТ слов в транскрипте
   - Причина: YouTube Content ID матчит оригинальный score сериала
     → "No restriction" claim в лучшем случае, бан в худшем

🚫 **ОБЫЧНЫЕ ЗАПРЕТЫ**
   - Рекламные вставки
   - Монотонные объяснения без эмоции/визуала
   - Фрагменты без законченной мысли
   - Технические заминки, паузы, мычание, «ээээ», «нууу», «так»
   - Повторяющиеся реплики типа «пока-пока-пока»
   - «Разогревы» — где интересное только в последние 5 секунд клипа
   - Сцены без чётко слышимой/видимой первой реплики

══════ START TIME — САМОЕ ВАЖНОЕ ПРАВИЛО ══════

`start` ДОЛЖЕН быть на МОМЕНТ ДЕЙСТВИЯ. НЕ за 2 секунды ДО реплики
«чтобы дать контекст» — это убьёт хук и swipe-away rate.

ПЛОХО: start=10.0 sec, а первая реплика на 12.0 sec → 2 сек тишины убьют видео
ХОРОШО: start=12.0 sec — сразу с реплики

══════ MUSIC MOOD (выбери одно по типу сцены) ══════

- "dramatic" — серьёзный момент, конфликт, драма, шок
- "hype" — драка, погоня, экшн, агрессия
- "sus" — напряжение, мистика, триллер, неловкость
- "chill" — спокойный разговор, размышления, романтика
- "upbeat" — энергия, динамика, активная сцена
- "comedy" — юмор, смех, абсурд, ирония
- "inspirational" — вдохновение, прорыв, мощная мысль
- "epic" — масштабная сцена, кульминация, эпическое событие

══════ ВРЕМЕННЫЕ ОГРАНИЧЕНИЯ ══════

ДЛИТЕЛЬНОСТЬ ВИДЕО: {total_seconds} секунд.
- Все start и end ОБЯЗАНЫ быть в диапазоне [0, {total_seconds}]
- start < end, end - start >= {min_sec}, end - start <= {max_sec}
- МОМЕНТЫ НЕ ДОЛЖНЫ ПЕРЕСЕКАТЬСЯ, минимум 10 сек между концом одного и началом следующего

КАК КОНВЕРТИРОВАТЬ ВРЕМЯ:
- Тайм-коды показаны как [MM:SS]. Формула: секунды = MM × 60 + SS
- [25:30] = 25 × 60 + 30 = **1530** секунд. НЕ 2530!

ВАЖНО: ты ОБЯЗАН вернуть РОВНО {clips_count} элементов. Не больше и не меньше.
Если кажется хороших моментов мало — снижай планку, но выдай {clips_count}.

══════ ПРОВЕРКИ ПЕРЕД ОТВЕТОМ ══════

1. clips содержит РОВНО {clips_count} элементов
2. Для каждого: {min_sec} <= (end - start) <= {max_sec}, идеал ~{target_sec}
3. start >= 0, end <= {total_seconds}, start < end
4. Клипы не пересекаются, разница между концом и следующим началом >= 10
5. На `start` секунде в транскрипте есть РЕПЛИКА, не тишина и не музыка
6. `hook` — это ТОЧНАЯ цитата из первых 2-3 секунд клипа
7. На `start` секунде есть РЕПЛИКА в транскрипте, не заставочная музыка
8. В транскрипте между start и end НЕ МЕНЕЕ 8 слов реплик (не "тра-ля-ля")
9. В первые 3 секунды клипа есть НАСТОЯЩИЕ слова реплики, не fill-музыка
